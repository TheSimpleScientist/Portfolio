The Front-End Web Developer Nanodegree is phase two of the Google Developer Scholarship Program. I was accepted into phase two of the program after successfully completing phase one (a 3 month challenge course on Udacity). The challenge course covered the following areas:

  • HTML syntax
  • CSS syntax
  • How to write code faster
  • JavaScript data types, variables, conditionals, loops, functions, arrays, and objects
  • DOM (Document Object Model)
  • jQuery

Only the top 10% from phase one were accepted into phase two of the program. The top 10% was determined based upon the participant's performance in the challenge course and their participation within the studenty community.
