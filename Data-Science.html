I am currently participating in phase one of the Bertelsmann Data Science Scholarship Program. Phase one is a 3 month challenge course on Udacity that covers descriptive statistics, Python, and SQL (Structured Query Language).

Descriptive statistics is the core subject of the data science challenge course. The key concepts on descriptive statistics covered are:

  • Statistical research methods
  • Computing and interpreting values (e.g. - mean, medium, mode, sample, population, standard deviation)
  • Computing probabilities
  • Data visualization (e.g. - bar graphs, histograms, box plots)
  • Investigate and understand distributions, and manipulate them to make predictions on data

Python is an advanced subject of the data science challenge course. The key concepts on Python covered are:

  • Fundamentals and best practices of the Python programming language
  • Represent and store data by using data types and variables
  • Control the flow of programs by using conditionals and loops
  • Store collections of related data by using complex data structures (e.g. - lists, sets, dictionaries, and tuples)
  • Define and document custom functions, write scripts, and handle errors
  • Find and use modules in the Python Standard Library and other 3rd party libraries

SQL is another advanced subject of the data science challenge course. The key concepts on SQL covered are:

  • Extract and analyze data stored in databases by using SQL
  • Join tables together and perform aggregations
  • Perform complex analysis and manipulations using subqueries, temp tables, and window functions
  • Write efficient SQL queries to successfully handle a variety of data analysis tasks

Participants in phase one will have the opportunity to move on to phase two of the program (a full 6 month nanodegree on Udacity) depending upon their performance in the challenge course and their participation within the studenty community. Hope to share more on phase two soon!
